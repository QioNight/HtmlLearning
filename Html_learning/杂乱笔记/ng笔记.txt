	AngularJS
MVC 模块化 双向数据绑定 指令

ng生命周期 简单分为 3 个阶段
	(每次浏览器加载web页面)
	1、启动：ng下载到浏览器，初始化所需组件，初始化根模块（ng-app），模块加载并可用
	2、编译：web页面 编译静态DOM加载浏览器 遍历静态DOM所有指令，把指令链接到内置ng库或自定义指令代码中，最后再生成实时视图
	3、运行：运行时数据绑定 数据模型绑定视图

MVC
	设计结构 职责划分
M(model) 数据模型（数据）
V(view) 视图（展示数据）
C(controller) 控制器（实现 M和V 交互）

model -> view
模板 + model + controller -> view
( {{表达式}} + $scope + js -> view )



模块
	一切都是从模块开始
	可以放
		控制器、指令、过滤器(数据显示格式)
		
	定义ng根元素 (ng-app)
